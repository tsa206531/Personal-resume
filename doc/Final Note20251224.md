# 個人履歷網頁技術分析報告 (2025-12-24)

## 一、簡單描述

本網頁是一個現代化的個人作品集與履歷系統，採用最新的 **Nuxt 3** 框架開發。其核心設計理念在於打造一個快速、流暢且具備跨平台響應能力的展示空間。透過整合 **Tailwind CSS** 與 **Vue 3** 的組合式 API（Composition API），網頁成功實現了精美的視覺動效與高效的資料管理模式。這不僅是一個靜態展示頁面，更是一個體現技術深度與美學追求的開發實踐，讓造訪者能快速掌握開發者的技術棧與專案成果。

## 二、功能描述

- **個人品牌展示**：首頁整合大頭照與簡潔的自我介紹，傳達開發者的專業定位與開發理念。
- **互動式技能輪播**：使用 Swiper 技術實作技能專長展示區，以動態橫向滑動方式呈現多樣化的技術標籤（如 Vue, Tailwind, GitHub, Cucumber 等）。
- **專案作品集管理**：透過高度組件化的方式展示多個開發專案，包含專案主題、描述與即時預覽連結，部分內容透過 iframe 整合外部託管的展示頁面。
- **履歷下載功能**：提供外部連結或 PDF 版本履歷的直接下載/查閱管道，便於求職需求。
- **主題色切換（深淺模式）**：系統支援深色/淺色模式切換，根據系統設定或使用者偏好自適應呈現，提升視覺舒適度。

## 三、完整技術描述

### 前端技術

- **核心框架**：**Nuxt 3**（基於 Vue 3），利用模組化架構與約定優於配置的特性提升開發效率。
- **開發語言**：**TypeScript**，為專案提供強類型檢查，有助於減少運行時錯誤並提升代碼可維護性。
- **樣式設計**：**Tailwind CSS**，採用 Utility-first 理念進行快速 UI 建構，並配合 **PostCSS** 與 **Autoprefixer** 處理 CSS 預處理與瀏覽器相容性。
- **視覺組件**：
  - **Swiper.js**：用於建立高效能、支援觸控的技能展示輪播。
  - **Heroicons / Simple Icons**：作為主要的向量圖示來源，確保介面視覺一致性。
  - **Google Fonts**：導入 "Zen Old Mincho" 等字體提升專案質感。
- **工具庫**：**@vueuse/core**，提供了豐富的 Composition API 工具函式，簡化了 DOM 監聽與狀態管理。

### 後端技術與架構

- **SSR/SSG 混合模式**：利用 Nuxt 的伺服器端渲染技術確保良好的 SEO 與首屏加載速度，並能無縫切換為靜態生成模式。
- **基礎架構**：專案遵循標準的 Nuxt 目錄結構（Layouts, Pages, Components），有效落實職責分離。
- **Server 資源**：目錄中雖包含 `server/` 資料夾，但目前主要作為配置環境（tsconfig），實體邏輯多集中於前端。

### 資料處理方式

- **組合式資料管理**：採用 **Composables (useSkills, useProjects)** 封裝靜態資料陣列，利用 Vue 3 的響應式系統（Ref）進行資料傳遞。
- **資源管理策略**：由於環境路徑限制，專案目前採用外部圖片託管（如 GitHub Raw），確保專案在不同部署平台（如 Vercel）上的穩定性。

### 開發模式與測試

- **行為驅動開發 (BDD)**：根據專案提示，開發過程中嚴格遵循 BDD 流程。這包括將功能描述轉化為驗收情境，並採用最小增量原則進行開發與驗證。
- **Clean Code 原則**：在開發過程中強調持續重構，確保代碼在功能通過測試後依然保持整潔且易於擴充。

## 四、未來可以改善的方向

1.  **資產優化處理**：建議導入 **Nuxt Image** 模組，這不僅能自動優化圖片格式與大小，還能解決 `useProjects.js` 提到的本地路徑載入問題。
2.  **動態內容管理**：目前資料雖然已抽離到 Composables，但仍屬硬編碼。未來可考慮與 **Headless CMS**（如 Contentful）或 **Content SDK** 串接，實現無需重新部署即可更新內容。
3.  **自動化測試補全**：專案已提及 Cucumber 與 BDD 概念，未來可實際導入 **Vitest** 或 **Cypress** 進行單元測試與端對端測試，確保核心流程的穩定性。
4.  **去 Iframe 化**：目前部分作品集採用 iframe 嵌入，這可能影響整體的 SEO 權重與使用者體驗。建議直接將該內容重構為專案內部的 Vue 組件，以獲得更好的效能與視覺整合感。
